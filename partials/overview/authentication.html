Authentication

Example Authentication Headers:

    X-MOXI-ID: 29025b28
    X-MOXI-SECRET: FavpXA

The Moxi Works API utilizes an API Identifier and Secret combination for Authentication and Authorization.

A X-MOXI-ID HTTP header containing your Moxi Works API Identifier is required to authenticate requests.

A X-MOXI-SECRET HTTP header containing your Moxi Works API Secret is required to authenticate requests.

You can acquire your Moxi Works API Identifier and Secret pair by contacting Moxi Works.

Do not include your Moxi Works API Identifier or Secret in the parameters sent to the server; they must be set in the request header. Setting the API Identifier and Secret in the request parameters will result in an authorization failure.
