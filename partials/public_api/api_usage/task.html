Send a GET request to https://api.moxiworks.com/api/tasks

When searching for a Task using the Moxi Works platform API, format your data using the following parameters.

Attribute Type Remark Length Limit
moxi_works_agent_id String The agent’s Moxi Works Platform ID 255
due_date_start Integer the Unix timestamp representing the start time. When searching using a date range, both due_date_start and due_date_end must be included. 255
due_date_end Integer the Unix timestamp representing the end time. When searching using a date range, both due_date_start and due_date_end must be included. 255
partner_contact_id String The unique ID of the contact in your system 255
page_number Integer Page of responses. Use if total_pages indicates that there is more than one page of data available. 255

Required Parameters Are In Red

If no parameters are included, all agent tasks will be returned.
