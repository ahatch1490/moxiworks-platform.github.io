Send a POST request to https://api.moxiworks.com/api/events

When creating an event using the Moxi Works platform API, format your data using the following parameters.

Event Creation Example

POST /api/events HTTP/1.1
Host: api.moxiworks.com
Accept: application/vnd.moxi-platform+json;version=1
Authorization: Basic XNNeXAAAUZaQlF0dA==
Content-Type: application/x-www-form-urlencoded


moxi_works_agent_id=abc123&partner_event_id=abc98df345&event_subject=subject&
event_location=a%20location&
is_meeting=true&all_day=false&event_start=1462234227&event_end=1462241468&
required_attendees=&optional_attendees=&note=here%20is%20the%20note

Attribute Type Remark Length Limit
moxi_works_agent_id String The agent’s Moxi Works Platform ID 255
partner_event_id String The unique ID of the event in your system 255
event_start Integer The Unix timestamp representation of the start time. 255
event_end Integer The Unix timestamp representation of the end time. 255
event_subject String A short description of the event 255
event_location String A description of the location of the event 255
attendees String A comma separated list of contacts that have already been added through the Moxi Works Platform API whose presence at event is required. (use IDs from your system – i.e. partner_contact_id from Contact Create ) who are to be included as required attendees.
note String a note to be included with additional details. Any HTML formatting included will be stripped in response.

Required Parameters Are In Red

Attendees must be Contacts with an associated email address. Attendees added who don’t have an email address associated will not be added to the event.
