Index will return a paged response of agents that have been updated since a given timestamp for a given company. For a list of company IDs that you have access to, use the Company Endpoint If no updated_at parameter is provided, agents that have been updated in the last seven days will be returned.

Agent Index Example

All Agents For Company Updated Since A Given Timestamp

GET /api/agents?moxi_works_company_id=abc123&updated_since=1461108284 HTTP/1.1

Authorization: Basic XNNeXAAAUZaQlF0dA==

Send a GET request to https://api.moxiworks.com/api/agents

When searching for Agent entities using the Moxi Works platform API, format your data using the following parameters.

Attribute Type Remark Length Limit
moxi_works_company_id String A valid Moxi Works Company ID. Use Company Endpoint to determine available moxi_works_company_id 255
updated_since Integer any agents updated after this Unix timestamp will be returned in the response. If no updated_since parameter is included in the request, only agents updated in the last seven days will be returned. 255
page_number Integer Page of responses to return. Use if total_pages indicates that there is more than one page of data available. 255

Required Parameters Are In Red

If no updated_since parameter is included in the request, only agents updated in the last seven days will be returned.
Use Company Endpoint to determine the moxi_works_company_id to send.
