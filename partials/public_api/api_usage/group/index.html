Send a GET request to https://api.moxiworks.com/api/groups

When searching for a Group using the Moxi Works platform API, format your data using the following parameters.

Attribute Type Remark Length Limit
moxi_works_agent_id String The agent’s Moxi Works Platform ID 255
name String the name of a group to be searched for. If no name parameter is included, all defined groups will be returned in the response.
