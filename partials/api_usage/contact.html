Contact

Creating

When creating a contact using the Moxi Works platform API, format your data using the following structures.

One of the following must be set when creating a new contact: first_name or last_name or primary_email_address or secondary_email_address or primary_phone_number or secondary_phone_number

Updating

One of the following must be set when updating a contact: first_name or last_name or primary_email_address or secondary_email_address or primary_phone_number or secondary_phone_number
Moxi Works API will only update the attributes that are defined in a given request. For instance, if the first_name attribute is not passed, the current first_name will be left as it is currently stored in Moxi Works API.
Attribute Type Remark Required
